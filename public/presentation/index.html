Problem

Обычный AI-чат часто даёт текст. Пользователю нужен готовый артефакт.

Слишком много ручной сборки

Нужно самому просить HTML, CSS, JS, адаптив, состояния, проверку и запуск.

Нет продуктового контекста

Код без MVP, оффера, сценариев и бизнес-логики плохо помогает запустить идею.

Solution

Amethyst объединяет разработку и запуск идеи в одном интерфейсе.

Code

React, TypeScript, HTML, CSS, review, debugging и объяснение ошибок.

Build

Лендинги, web-app MVP, playable games и HTML artifacts для скачивания.

Business

Оффер, структура продукта, план продаж, презентация и первые шаги запуска.

Two platforms

Две платформы: desktop studio и mobile mode.

DesktopПолная студия: чат, файлы, история, генерация и скачивание HTML.
MobileБыстрые режимы, QR-вход, адаптивный composer и работа на телефоне.
ArtifactsСайты, MVP, игры, компоненты, bug fix и бизнес-brief.
GeminiAI вызывается через Supabase Edge Function, ключ не на фронте.
